# Welcome to Cyclops Documentation

Cyclops is a collection of tools and playbooks for deploying, managing, and operating Kubernetes clusters.

This site contains documentation for all repositories in the [cyclops-k8s](https://github.com/cyclops-k8s) organization.

## Repositories

| Repository | Description |
|-----------|-------------|
| [Ansible Kubernetes](ansible-kubernetes/index.md) | CIS/STIG hardened Kubernetes cluster deployment via Ansible |
| [Devcontainer](ansible-kubernetes-devcontainer/index.md) | VS Code devcontainer for the ansible-kubernetes project |
| [Build Agent Image](build-agent-image/index.md) | Custom GitHub Actions Runner image for CI/CD |
| [Cyclops Assets K8s](cyclops-assets-k8s/index.md) | Kubernetes manifests for asset serving and registry mirrors |
| [RadosGW Usage Exporter](radosgw-usage-exporter/index.md) | Prometheus exporter for Ceph RGW usage metrics |
| [Squid Container](squid-container/index.md) | Squid caching proxy with SSL BUMP support |
| [Swiss Army Knife](swiss-army-knife/index.md) | Networking and diagnostic toolbox for Kubernetes |
| [MultiCluster Ingress](multicluster-ingress/index.md) | Multi-cluster Kubernetes DNS and GSLB operator |
