Welcome to Cyclops Documentation¶

Cyclops is a collection of tools and playbooks for deploying, managing, and operating Kubernetes clusters.

This site contains documentation for all repositories in the cyclops-k8s organization.

Repositories¶

Repository Description
Ansible Kubernetes CIS/STIG hardened Kubernetes cluster deployment via Ansible
Devcontainer VS Code devcontainer for the ansible-kubernetes project
Build Agent Image Custom GitHub Actions Runner image for CI/CD
Cyclops Assets K8s Kubernetes manifests for asset serving and registry mirrors
RadosGW Usage Exporter Prometheus exporter for Ceph RGW usage metrics
Squid Container Squid caching proxy with SSL BUMP support
Swiss Army Knife Networking and diagnostic toolbox for Kubernetes
MultiCluster Ingress Multi-cluster Kubernetes DNS and GSLB operator
